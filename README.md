# alpha-camp-vue
簡易電商購物網站

## 功能
* 點擊下一步＆上一步可切換頁面步驟
* 可切換運費價錢
* 可以增加或減少商品數量，金額會隨著數量而更動

## Project setup
```
npm install
```
